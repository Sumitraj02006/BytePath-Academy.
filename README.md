# BytePath Academy Landing Page

A premium, modern, Gen-Z friendly, and responsive landing page for BytePath Academy.

## Features
- **Responsive Navigation**: Sticky navbar with responsive slide-out hamburger menu.
- **Vibrant Gradients & Ambient Lighting**: Midnight-blue background theme combined with royal blue and purple backlights.
- **Glassmorphic Design**: Semi-transparent card styling with borders and backdrop blur.
- **Interactive Roadmap**: Connected timeline showing Level 1 to 4 courses.
- **Lead Generation Form**: Complete inline form validation and live database sync with Google Sheets.
- **Innovation Showcase**: Hackathon results poster display.
- **Social Media Hub**: Custom branded hovering glow cards for LinkedIn, Instagram, Facebook, and X.

## Technology Stack
- **HTML5** & **CSS3** (Fluid layout, responsive grids, custom SVG icons)
- **JavaScript** (Intersection Observers, programmatic smooth scrolling, Fetch API integration)
- **Google Apps Script** (Endpoint listener for form submissions)

## How to Run Locally
1. Clone this repository.
2. Open `index.html` in any web browser.
